# 🧠 Deepu's Copilot Agents

Welcome to **Deepu’s custom GitHub Copilot agents** — a collection of specialized AI assistants designed to improve developer productivity, scale engineering workflows, and bring expert‑level knowledge directly into your coding environment.

This repository contains AI agents written in the **`.agent.md`** format and can be installed directly into **VS Code Copilot Chat**.

---
